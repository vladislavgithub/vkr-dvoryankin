## Структура

```
main.tex                    — корневой файл
chapters/
  00-intro.tex              — Введение
  01-analysis.tex           — Глава 1. Анализ предметной области
  02-models.tex             — Глава 2. Модели взаимодействия
  03-implementation.tex     — Глава 3. Проектирование и реализация сервиса
  04-reliability.tex        — Глава 4. Обеспечение надёжности и безопасности
  05-experiment.tex         — Глава 5. Экспериментальное исследование
  06-conclusion.tex         — Заключение
bib/sources.bib             — Список литературы
figures/                    — Иллюстрации
```

## Сборка PDF

```bash
docker compose run --rm latex   # локально через TeX Live в docker
# или
make
```

Каждый push в `main` запускает GitHub Actions: PDF собирается и публикуется как артефакт + GitHub Release.
